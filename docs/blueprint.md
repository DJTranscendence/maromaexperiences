# **App Name**: Maroma Experiences

## Core Features:

- Individual Booking: Allow individual users to book spaces in group tours and workshops, with a real-time display of booked and free spaces, indicated by color-coded status.
- School Booking: Enable schools to book tours and workshops for students, integrated seamlessly into the existing booking system.
- Corporate Booking: Offer curated corporate experience packages with optional add-ons via drop-downs and checkboxes for a customized booking experience.
- User Account Creation: Allow users to create and manage accounts with name, email, and phone number, ensuring a personalized booking experience.
- Admin Tour Management: Provide an admin interface for creating, editing, and managing tours and workshops, including scheduling and capacity settings.
- Smart Notifications Tool: A notification service leveraging generative AI. Gemini will generate personalized messages for booking-related events and intelligently decide when the user wants to incorporate links to external tools (email, WhatsApp, Telegram)

## Style Guidelines:

- Primary color: Deep blue (#2E4A62) to convey trust and professionalism in booking services.
- Background color: Light gray (#EAEAEA), providing a clean, neutral backdrop that highlights the tour and workshop details.
- Accent color: Warm orange (#D97A30) to draw attention to key interactive elements.
- Body font: 'League Spartan', a clean sans-serif for an elegant user experience.
- Headline font: 'Alegreya', serif, lending elegance to headlines and important titles.
- A consistent set of minimalist icons representing tours, workshops, locations, and activities.
- A clean, spacious layout with clear hierarchy, emphasizing visual elements like high-quality photos and videos.
- Subtle transitions and animations to enhance user interaction without being distracting.
